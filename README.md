# 图灵计划-系统开发计划与功能点梳理


## 系统开发计划

## I. 调研结束 (2018. 4.1 - 2018. 5. 15) 1.5 个月
调研部分主要分为，GO 高级特性总结，区块链系统架构分析，该系统与一般区块链
系统比较的特殊性分析，完成相关文档的编写整理和对外发布
## II. 需求分析(2018. 5.1 - 2018. 6.1) 1 个月
对图灵计划的整体系统需求分析，整理出系统所需要的功能点，进行所有功能的可行
性个研究，完成系统需求说明书
## III. 系统设计(2018. 6.1 - 2018. 7.1) 1 个月
完成系统的主体设计，完成详细的算法设计，完成可对外发布的系统设计说明书
## IV. 编码阶段(2018. 7.1 - 2018. 9.30) 3 个月
编码正式开始，陆续开始分阶段提交 Github，实现所有算法与功能
## V. 系统测试(2018. 9.15 - 2018. 11.30) 1.5 个月
对最终的系统进行模块测试，集成测试及 bug 修复，完成最终的测试报告和系统的安
全评估报告
